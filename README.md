# Oracle HCM SDK

This gem is a library for interacting with the Oracle HCM REST API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'oracle_hcm'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install oracle_hcm
